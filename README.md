# 给妈妈的 3·8 妇女节网页

本项目是一个简单的单页静态网页，用来向妈妈发送节日祝福。打开 `index.html` 即可在浏览器中查看。

快速使用：

1. 在文件管理器中打开 `index.html`（双击或在浏览器中打开）。
2. 若要更换照片，请把图片放到 `assets/mom-photo.jpg`（或修改 HTML 中的 `src` 指向您的文件）。
3. 点击“给妈妈一个惊喜 ✨”按钮会触发彩带与爱心动画。
4. 点击“生成微信二维码”会生成当前页面地址的二维码，方便用微信扫码打开（若页面尚未部署到网上，请先发布到可访问地址再分享二维码）。

如需我帮忙：
- 我可以替您替换文字、配色或写一段更长的祝福语。
- 我可以把页面打包、生成可分享的单文件（包含图片/音乐），或者帮您部署到 GitHub Pages。
